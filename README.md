# PreCheck

## Beskrivning
PreCheck är en tjänst som kontrollerar möjligheten att leverera olika tjänster för en specifik adress.

## Tekniska detaljer

### Integrationer
Tjänsten integrerar mot:

* Lantmäteriverkets API för belägenhetsadress
* GIS

## Leverantör

Sundsvalls kommun
